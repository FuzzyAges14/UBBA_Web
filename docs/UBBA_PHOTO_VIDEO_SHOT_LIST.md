# UBBA Photo & Video Shot List

**Purpose:** Replace temporary stock with authentic United Black Belt Academy media for Allendale, Midland Park, and Glen Rock (when enabled).  
**Audience:** Owner, instructors, hired photographer/videographer.  
**Related:** [`MEDIA_AUDIT.md`](MEDIA_AUDIT.md), [`MEDIA_CANDIDATES.md`](MEDIA_CANDIDATES.md), [`PERFORMANCE.md`](PERFORMANCE.md)

---

## 1. Goals

Capture media that feels:

- Premium and cinematic, but family-friendly  
- Clearly **Taekwondo / UBBA**, not generic karate or MMA  
- Warm enough for parents; energetic enough for teens  
- Usable on **desktop heroes (16:9)** and **mobile posters (center-weighted / 9:16)**  
- Safe to publish with model releases (especially minors)

Do **not** stage frightening contact, blood, or fight-promo lighting.

## 3. Technical specs

### Still photography

| Spec | Recommendation |
| --- | --- |
| Camera | Full-frame or APS-C mirrorless; sharp primes/zooms |
| Resolution | Deliver ≥ 4000px on long edge |
| Formats | RAW + edited JPEG/WebP masters |
| Color | Consistent white balance; slightly warm skin; deep blacks OK |
| Deliverables | Hero 1920×1080, cards 1280×720, portraits 1200×1600, OG 1200×630 |
| Avoid | Heavy HDR halos, fake sharpening, muddy ISO noise |

### Video

| Spec | Recommendation |
| --- | --- |
| Resolution | 4K acquisition; master 1080p delivery for web |
| Frame rate | 24/30 fps primary; optional 60/120 for slow-mo kicks |
| Audio | Room tone OK; **website hero is muted** — prioritize picture |
| Hero loop | 8–15 seconds, seamless or soft fade, no text burned in |
| Encode targets | MP4 H.264 ≤ ~3 MB; WebM VP9 companion (see `PERFORMANCE.md`) |
| Vertical | Capture 9:16 social cuts in same session |

### Lighting

- Soft key + gentle fill (window + LED panels)  
- Avoid pure black void studio (current Mixkit look)  
- Keep mat texture and red/blue brand accents visible when natural

## 8. What not to shoot / deliver

- MMA cage, mouthpiece close-ups, anger-forward fight faces  
- Stock-style pure black void with single posing fighter (current placeholder look)  
- Unreleased minors  
- Other schools’ logos or tournament backdrops that confuse brand  
- Heavy filters that break across the site palette  
- Watermarked proofs as production files
