## 5. Martial-arts accuracy checklist (applied)

| Check | Finding |
| --- | --- |
| Uniform plausibility | Prefer white dobok; reject casual streetwear “karate” memes |
| Belt presentation | Avoid incorrectly tied / fashion belts; cottonbro set generally OK |
| Technique | Prefer pads, forms, bows over flashy spinning cinema kicks |
| Protective equipment | Soft gear OK for teens; avoid MMA gloves/cage |
| Class environment | Prefer mats + mirrors + dojang over black void studio |
| Instructor behavior | Coaching/bowing > intimidating stare |
| Age appropriateness | Match Tiny Tigers vs teen vs adult slots |
| Competitor branding | Reject visible school logos on doboks when possible |
| Cultural accuracy | Taekwondo Korean heritage OK; don’t label karate as TKD in alt text |
| AI-generated people | Avoid for people photography (Pixabay AI illustrations rejected) |
